# Customer-Churn-Analysis-Power-BI-Project
## 📌 Project Overview

This project analyzes customer churn behavior using a banking dataset.  
The objective is to identify high-risk customer segments and understand the key drivers of churn through data visualization and segmentation analysis in Power BI.

The analysis focuses on behavioral patterns, demographics, and product usage to support data-driven retention strategies.

---

## 🎯 Business Problem

Customer retention is critical for profitability in banking.  
This project aims to answer:

- What is the overall churn rate?
- Which customer segments are at highest risk?
- How do demographics and product usage influence churn?

---

## 📂 Dataset Information

**Dataset Columns Used:**

- CreditScore  
- Geography  
- Gender  
- Age  
- Tenure  
- Balance  
- NumOfProducts  
- HasCrCard  
- IsActiveMember  
- EstimatedSalary  
- Exited (Churn Indicator)

> Exited = 1 indicates the customer left the bank.

## 📈 Key Metrics

- **Total Customers:** 10,000  
- **Churned Customers:** 2,000+  
- **Overall Churn Rate:** 20.37%  

---

## 🔍 Key Insights

- Germany has the highest churn rate (32.44%).
- The 50–59 age group shows the highest churn rate (56%).
- Customers using 4 products have the highest churn probability.
- Active customers demonstrate significantly lower churn rates.
- Churn behavior shows non-linear relationships across segments.
